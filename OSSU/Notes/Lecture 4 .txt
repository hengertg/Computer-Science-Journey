# 📌 Notes - The `count` Variable in Python

## What does `count` do?

`count` is a variable used to **count how many times a specific event occurs** in a program.

It usually starts at `0`.

```python
count = 0
```

Every time the condition we are looking for is true, we increase the counter by one.

```python
count += 1
```

This is exactly the same as writing:

```python
count = count + 1
```

---

## When should I use `count`?

Use `count` whenever you need to answer questions like:

* How many even numbers are there?
* How many people are over 18?
* How many errors occurred?
* How many times does a letter appear?

Example:

```python
count = 0

for number in numbers:
    if number % 2 == 0:
        count += 1

print(count)
```

Every time an even number is found, the counter increases by one.

---

## MIT Example

```python
seen = ""
count = 0

for char in s:
    if char not in seen:
        seen += char
        count += 1

print(count)
```

Here, `count` increases only when a **new unique character** is found.

---

## My Observation (Henger)

In this exercise, using `count` was **not necessary**.

Since the string `seen` only stores unique characters, I can get the same result by using:

```python
print(len(seen))
```

Why?

* `seen` contains only unique characters.
* `len(seen)` returns the total number of characters stored.

My solution:

```python
seen = ""

for char in s:
    if char not in seen:
        seen += char

print(f"There are {len(seen)} unique letters.")
```

This solution removes an unnecessary variable and makes the code cleaner.

---

## Conclusion

I will use `count` when **there is no other simple way to determine the quantity**.

If I already have a data structure (such as a string, list, or set) that contains exactly the elements I need to count, I should first consider using `len()` instead of creating an additional counter.

### Important Question to Ask Myself

> **"Do I already have a data structure that contains exactly what I want to count?"**

If the answer is **yes**, then `len()` is probably enough.

Otherwise, using a `count` variable is the correct approach.
